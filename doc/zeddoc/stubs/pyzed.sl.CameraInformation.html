CameraInformation¶

class pyzed.sl.CameraInformation¶

Structure containing information of a signle camera (serial number, model, calibration, etc.)

Those information about the camera will be returned by get_camera_information().

Note

This object is meant to be used as a read-only container, editing any of its field won’t impact the SDK.

Attributes

calibration_parameters

Intrinsic and Extrinsic stereo parameters for rectified/undistorded images (default).

calibration_parameters_raw

Intrinsic and Extrinsic stereo parameters for original images (unrectified/distorded).

camera_imu_transform

IMU to Left camera transform matrix, that contains rotation and translation between IMU frame and camera frame.

camera_model

The model of the camera (ZED or ZED-M).

firmware_version

The internal firmware version of the camera.

serial_number

The serial number of the camera.

Those information about the camera will be returned by get_camera_information().

Note

This object is meant to be used as a read-only container, editing any of its field won’t impact the SDK.

calibration_parameters¶

Intrinsic and Extrinsic stereo parameters for rectified/undistorded images (default).

calibration_parameters_raw¶

Intrinsic and Extrinsic stereo parameters for original images (unrectified/distorded).

camera_imu_transform¶

IMU to Left camera transform matrix, that contains rotation and translation between IMU frame and camera frame. Note that this transform was applied to the fused quaternion provided in get_imu_ata() in v2.4 but not anymore starting from v2.5. See get_imu_data() for more info.

camera_model¶

The model of the camera (ZED or ZED-M).

firmware_version¶

The internal firmware version of the camera.

serial_number¶

The serial number of the camera.
